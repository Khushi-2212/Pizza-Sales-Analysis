# Pizza Sales Analysis (SQL & Power BI)

## Project Overview
This project provides a comprehensive analysis of pizza sales data to help a business understand its performance. It involves data processing in **SQL Server** and visualization in **Power BI**.

## Key Questions Answered:
- What is the total revenue and average order value?
- Which days and times have the highest order volume?
- What are the top and bottom 5 selling pizzas?
- How do sales vary across different pizza categories and sizes?

## Tech Stack Used:
- **SQL Server:** For data cleaning and KPI calculation.
- **Power BI:** For creating interactive dashboards.
- **DAX:** For advanced analytical measures.

## How to Run:

1. Import the `pizza_sales.csv` into SQL Server.
2. Run the queries provided in the `SQL_Queries/` folder.
3. Open the `.pbix` file in Power BI Desktop to view the report.
